# 🧹 Чистильщик ссылок

Инструмент для очистки URL от трекеров и проверки безопасности ссылок.

## 🚀 Возможности

- Удаление UTM-меток и трекеров (fbclid, gclid и др.)
- Проверка доменов по чёрному списку
- Раскрытие коротких ссылок
- Проверка HTTPS
- Веб-интерфейс с реальным временем

## 📦 Установка

```bash
git clone https://github.com/ВАШ_НИК/url-cleaner.git
cd url-cleaner
pip install -r requirements.txt
python app.py
